# Contributing — SM-A022F/DS GSI Guide

Thanks for helping improve this guide.

---

## What You Can Contribute

- Fix typos or formatting errors
- Add missing troubleshooting entries
- Improve or clarify existing steps
- Add tested GSI compatibility notes

---

## How to Contribute

1. Fork the repository
2. Create a branch: `fix/your-topic` or `add/your-topic`
3. Make your changes
4. Submit a pull request with a short description of what you changed

---

## Guidelines

- Keep language simple and direct
- Test steps on a real SM-A022F/DS before submitting
- Do not add steps for other models (A02s, A03, etc.)
- Follow the existing file structure and heading style

---

## Reporting Issues

If you find an error but don't want to submit a fix, open an issue with:

- Which file and section has the problem
- What is wrong
- What the correct information should be
